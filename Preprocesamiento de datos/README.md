# 🧹 Desafío VII: Preprocesamiento de Datos y Modelado Predictivo

En este proyecto se trabaja sobre el preprocesamiento de datos y la construcción de modelos predictivos utilizando técnicas de Machine Learning en Python.  
El enfoque principal está puesto en la limpieza, transformación y preparación de los datos para mejorar el desempeño de los modelos.

---

## 📌 Contenido:

### 🔹 Análisis exploratorio de datos (EDA)

- Exploración inicial del dataset.
- Identificación de valores faltantes y registros duplicados.
- Análisis de distribuciones y relaciones entre variables.
- Visualización de patrones y detección de valores atípicos.

---

### 🔹 Preprocesamiento

- Limpieza y depuración de datos.
- Tratamiento de valores nulos.
- Eliminación de duplicados.
- Transformación de variables categóricas.
- Escalado y normalización de variables numéricas.
- Codificación de variables para entrenamiento de modelos.

---

### 🔹 Ingeniería de variables

- Creación y transformación de features.
- Reducción de dimensionalidad.
- Selección de variables relevantes.
- Evaluación de correlaciones entre variables.

---

### 🔹 Modelado

Se entrenan distintos modelos de Machine Learning para analizar el comportamiento de los datos:

- División en conjuntos de entrenamiento y prueba.
- Entrenamiento de modelos predictivos.
- Generación de predicciones.
- Comparación de desempeño entre modelos.

---

### 🔹 Evaluación del modelo

Se utilizan métricas para evaluar la calidad de las predicciones:

- R² (Coeficiente de determinación).
- MAE y MAPE.
- Comparación de errores entre modelos.
- Interpretación de resultados y capacidad predictiva.

---

## 🛠️ Tecnologías utilizadas:

- Python 3
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 🎯 Objetivo:

Aplicar técnicas de preprocesamiento y preparación de datos para construir modelos predictivos más robustos y comprender el impacto que tiene la calidad de los datos en el rendimiento del modelo.

---

## 📊 Resultados:

Se logró preparar el dataset mediante distintas técnicas de limpieza y transformación, permitiendo entrenar modelos predictivos y comparar su desempeño a través de métricas de evaluación.
