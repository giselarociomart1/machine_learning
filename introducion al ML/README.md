# 🧠 Desafío: Introducción al Machine Learning

Este repositorio contiene la resolución de un desafío introductorio de Machine Learning, donde se abordan conceptos teóricos fundamentales y una primera aproximación práctica utilizando Python.

---

## 📌 Contenido:

El proyecto está dividido en dos partes:

### 🔹 Parte I: Conceptos teóricos

Se responden preguntas clave sobre:

- Diferencias entre aprendizaje supervisado y no supervisado.  
- Aplicaciones de Machine Learning en la industria (caso: banca).  
- Uso de modelos de clasificación y regresión.  
- Importancia de dividir los datos en entrenamiento y prueba.  
- Diferencias entre enfoque estadístico y enfoque de ML.  

---

### 🔹 Parte II: Implementación práctica

Se trabaja con un dataset de viviendas (`data_housing.csv`) utilizando:

- `pandas` → manipulación de datos.  
- `numpy` → cálculos numéricos.  
- `scikit-learn` → modelos de machine learning.  

#### Tareas realizadas:

- Carga y exploración del dataset.  
- Preprocesamiento de datos.  
- Construcción de un modelo de regresión lineal.  
- Evaluación del modelo.  

---

## 🛠️ Tecnologías utilizadas

- Python 3  
- Pandas  
- NumPy  
- Scikit-learn  
- Jupyter Notebook  

---

## 🎯 Objetivo

Comprender los conceptos básicos de Machine Learning y aplicarlos en un caso práctico sencillo, integrando teoría y práctica.
