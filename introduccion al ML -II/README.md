# 🧠 Desafío II: Modelos de Machine Learning

Este proyecto corresponde al segundo desafío práctico de Machine Learning, donde se profundiza en el uso de modelos y en el flujo completo de trabajo: desde el análisis exploratorio hasta la evaluación de resultados.

---

## 📌 Contenido

El desarrollo incluye:

### 🔹 Análisis exploratorio de datos (EDA)

- Inspección del dataset.  
- Identificación de variables relevantes.  
- Detección de valores nulos y outliers.  
- Visualización de datos.  

---

### 🔹 Preprocesamiento

- Limpieza de datos.  
- Transformación de variables categóricas.  
- Escalado de variables (si aplica).  
- Separación en conjuntos de entrenamiento y prueba.  

---

### 🔹 Modelado

Se implementan modelos de Machine Learning utilizando `scikit-learn`, pudiendo incluir:

- Modelos de clasificación (ej: Logistic Regression, Decision Tree, etc.).  
- Ajuste del modelo con datos de entrenamiento.  

---

### 🔹 Evaluación

- Predicciones sobre el conjunto de prueba.  
- Métricas de evaluación (accuracy, precision, recall, etc.).  
- Interpretación de resultados.  

---

## 🛠️ Tecnologías utilizadas

- Python 3  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib / Seaborn  
- Jupyter Notebook  

---

## 🎯 Objetivo

Aplicar un flujo completo de Machine Learning, incorporando buenas prácticas de análisis, preprocesamiento y evaluación de modelos.

---

## 📊 Resultados

Se obtienen métricas que permiten evaluar el desempeño del modelo y entender su capacidad de generalización sobre datos no vistos.
