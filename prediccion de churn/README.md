# 🧠 Desafío III: Evaluación y Optimización de Modelos

En este proyecto se avanza sobre el desarrollo de modelos de Machine Learning, incorporando técnicas de evaluación más robustas y estrategias de optimización para mejorar el rendimiento.

---

## 📌 Contenido:

### 🔹 Análisis exploratorio de datos (EDA)

- Comprensión de la estructura del dataset.  
- Análisis de distribuciones y relaciones entre variables.  
- Identificación de posibles problemas en los datos.  

---

### 🔹 Preprocesamiento

- Limpieza y tratamiento de valores faltantes.  
- Codificación de variables categóricas.  
- Escalado de variables numéricas.  
- División en conjuntos de entrenamiento y prueba.  

---

### 🔹 Modelado

Se entrenan distintos modelos de Machine Learning, permitiendo comparar su desempeño:

- Modelos de clasificación o regresión según el problema.  
- Entrenamiento con datos de entrenamiento.  
- Generación de predicciones.  

---

### 🔹 Evaluación del modelo

Se aplican métricas más completas para medir el rendimiento:

- Accuracy, Precision, Recall, F1-score (clasificación). 
- MAE, MSE o RMSE (regresión).  
- Análisis de overfitting y underfitting.  

---

### 🔹 Optimización

- Ajuste de hiperparámetros.  
- Uso de técnicas como Grid Search o Random Search.  
- Mejora del rendimiento del modelo.  

---

## 🛠️ Tecnologías utilizadas

- Python 3  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib / Seaborn  
- Jupyter Notebook  

---

## 🎯 Objetivo

Mejorar la calidad de los modelos de Machine Learning mediante evaluación rigurosa y técnicas de optimización, entendiendo el impacto de cada decisión en el rendimiento final.


---

## 📊 Resultados

Se logra comparar distintos modelos y configuraciones, identificando cuál ofrece el mejor desempeño según las métricas seleccionadas.
